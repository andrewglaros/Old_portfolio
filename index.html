Projects

Boston Public Library redesign

As an exercise, I redesigned www.bpl.org. I wrote the code, created a responsive design, and improved the information architecture.

Simple Timer

I needed a simple exercise timer — so I built one. The five buttons in Simple Timer are common rest intervals for anaerobic workouts. Simple Timer adapts to any screen size.

Unbiased

Unbiased is a news app designed to encourage information gathering from multiple sources. The app provides two articles on one topic; sources are revealed when articles are read to completion.

LEED Online

LEED Online is USGBC’s web app that facilitates LEED project certification. I used USGBC’s existing style guide to craft a new user experience.
